# Objektai katalogai

Šiame vault'e pirminė objektų vieta yra `objektai/`.

Kategorijos:
- `objektai/saltiniai`
- `objektai/autoriai`
- `objektai/asmenys`
- `objektai/ivykiai`
- `objektai/daiktai`
- `objektai/paprociai`
- `objektai/posakiai`
- `objektai/zodynas`
- `objektai/vietos`
- `objektai/grupes`

Pagal dabartinę struktūrą ankstesni `entities/*` keliai nebenaudojami.
